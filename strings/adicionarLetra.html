Cree una función que coloque el primer argumento, un carácter, entre cada palabra del segundo argumento, una cadena.

  • add("R", "python is fun") ➞ "pythonRisRfun"

  • add("#", "hello world!") ➞ "hello#world!"

  • add("#", " ") ➞ "#"

  • El método join() devuelve la matriz como una cadena. Los elementos estarán separados por un separador especificado. El separador predeterminado es la coma (,) en este ejemplo seria el parámetro carácter.

  • El método split () se usa para dividir una cadena en una matriz de subcadenas y devuelve la nueva matriz. Si se utiliza una cadena vacía ("") como separador, la cadena se divide entre cada carácter. El método split () no cambia la cadena original.
